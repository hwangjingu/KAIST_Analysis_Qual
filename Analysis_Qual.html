1.
Prove that the set of x ∈ ℝ such that there exist infinitely many fractions p∕q, with relatively prime integers p and q such that
|x −p q | ≤ 1 q3

is a set of (Lebesgue) measure zero.

Sol.
See [5], p. 46 Problem 1. Because the Lebesgue measure μ is shift invariant, we only consider the case in the interval [0, 1]. We will denote p⊥q for relatively prime integers p and q > 0, or p = 0.

Give an enumeration on the set of all rationals in [0, 1]. Denote the n-th element as pn∕qn, where pn⊥qn. Let En = (pn∕qn − 1∕qn3,p n∕qn + 1∕qn3). Then

μ (⋃ nEn) ≤∑ nμ(En) = ∑ k ∑ n:qn=kμ(En) ≤∑ k2(k + 1) k3 < ∞,

where the second inequality is from that 0 ≤pn ≤qn. Then we have

μ (⋂ N≥1 ⋃ n≥NEn) = lim ⁡ N→∞μ (⋃ n≥NEn) ≤ lim ⁡ N→∞∑ n≥Nμ(En) = 0.

If x ∈ [0, 1] has infinitely many such fractions, then x ∈⋂ ⁡ N≥1 ⋃ n≥NEn, since for each q, at most one p with p⊥q can satisfy |x −p∕q|≤ 1∕q3. By countable additivity, the set of such x has zero measure.

1.
Let A ⊂ ℝ be a Lebesgue measurable set whose Lebesgue measure is strictly positive. Prove that there exists B ⊂A such that B is not Lebesgue measurable.
Sol.
See [5], p. 44 Exercise 32(b). It is just imitation to construct Vitali’s set.

Without loss of generality, assume A has finite measure. Since A has nonzero measure, there is an interval I such that A ∩I has nonzero measure. It is known that the set of all representives of the quotient group ℝ∕ℚ is nonmeasurable, where all representives are in I. Let N be such set.

Then A ∩N ⊂A ∩I is nonmeasurable; let {qk} be an enumeration of all rationals in I. Clearly A ∩I ⊆⊔ ⁡ k(A ∩ (N + qk)). By the way, there is an interval J such that

J ⊇⊔ kN + qk ⊇I.

If A ∩N were measurable,

0 < μ(A ∩I) ≤μ (⊔ k(A ∩ (N + qk))) = ∑ kμ(A ∩ (N + qk)) = ∑ kμ(A ∩N) ≤μ(A ∩J) < ∞

and A ∩N would have zero measure. But then μ(A ∩I) = 0, a contradiction.

Suppose that M is a σ-algebra in a set X and μ a finite measure on (X,M). We say that a sequence of measurable functions {fn}→f in measure if for every 𝜀 > 0
μ({x : |fn(x) −f(x)|> 𝜀}) → 0

as n →∞.

(a)
Prove that if fn →f almost everywhere (with respect to μ) then fn →f in measure.
(b)
Prove that if fn →f in measure then {fn} has a subsequence that converges to f almost everywhere (with respect to μ).
Sol.
(a)
Fix 𝜀 > 0. Let En := {x : |fn(x) −f(x)|> 𝜀}. Since the sequence {fn} converges to f almost everywhere, the set
⋂ N≥1 ⋃ n≥NEn

has zero measure; x is in the set if and only if for any N ≥ 1, |fn(x) −f(x)|> 𝜀 for some n ≥N, i.e., fn(x) does not converge to f(x).

As μ is finite, we can deduce that

μ (⋂ N≥1 ⋃ n≥NEn) = lim ⁡ N→∞μ (⋃ n≥NEn) ≤ lim ⁡ N→∞∑ n≥Nμ(En) = 0

and hence μ(En) → 0, i.e., fn converges to f in measure.

(b)
Let E𝜀,n := {x : |fn(x) −f(x)|> 𝜀}. Then there exists an increasing subsequence nk such that μ(E2−k,nk) ≤ 2−k. Then
μ (⋃ k=1∞E 2−k,nk) ≤∑ k=1∞μ(E 2−k,nk) ≤∑ k=1∞2−k < ∞

and

μ (⋂ N=1∞⋃ k=N∞E 2−k,nk) = lim ⁡ N→∞μ (⋃ k=N∞E 2−k,nk) ≤ lim ⁡ N→∞∑ k=N∞μ(E 2−k,nk) = 0.

For x∉⋂ ⁡ N=1∞⋃ k=N∞E 2−k,nk, there exists some N ≥ 1 such that |fnk(x) −f(x)|≤ 2−k for all k ≥N. That is, fnk(x) →f(x). Hence if fnk →f fails, then x ∈⋂ ⁡ N=1∞⋃ k=N∞E 2−k,nk, which has zero measure, i.e., fnk →f almost everywhere.

∥f∥p = sup ⁡ { |∫ Sfgdμ| : g ∈Lq(S,μ),∥g∥ q = 1} .

By Hölder’s inequality, if ∥g∥q = 1, then
|∫ Sfgdμ| ≤ (∫ S|f|pdμ)1∕p (∫ S|g|qdμ)1∕q = ∥f∥ p

1.
Let f : ℝm → ℝn be a continuous mapping. Prove that, if A is a Borel subset of ℝn, then f−1(A) is a Borel subset of ℝm.
Sol.
Since f is continuous, f−1 preserves openness and closedness. Hence if A is either open or closed in ℝn, so is f−1(A) in ℝm.

Furthermore, the inverse image preserves unions and intersections. As a Borel set is generated by countable unions and intersections of open and closed sets, if A is a Borel subset, so is f−1(A).

2.
Prove the following:
(a)
There exists a positive continuous function f on ℝ so that f is integrable on ℝ, but limsup ⁡ x→∞f(x) = ∞.
(b)
If f is uniformly continuous on ℝ and integrable, then lim ⁡ |x|→∞f(x) = 0.
Sol.
(a)
Let g(x) = exp ⁡ (−x2), which is continuous, integrable on ℝ, and positive. Let
h(x) = (−|x| + 1)1|x|≤1.

For each k ∈ ℤ, let

hk(x) = 2|k|h(4|k|(x −k)).

Finally, define

f(x) = g(x) + ∑ k∈ℤ∖{0}hk(x),

where the series is well defined; for each l ∈ ℤ ∖{0} and x ∈ (l − 4|l|,l + 4|l|), we have

∑ k∈ℤhk(x) = hl(x).

Then

∫ f = ∫ g + ∫ ∑ khk = ∫ g + ∑ k ∫ hk = ∫ g + ∑ k2−|k|+1 < ∞.

However, for each n ∈ ℕ, f(n) > hn(n) = 2n and hence limsup ⁡ x→∞f(x) = ∞.

(b)
Suppose f does not vanish at infinity. Then there exist 𝜀 > 0 and a sequence {xn} with xn + 1 < xn+1 such that |f(xn)|> 𝜀. For such 𝜀, there exists 1 > δ > 0 such that |x −y|< δ implies |f(x) −f(y)|< 𝜀∕2. Which means that |x −xn|< δ implies |f(x)|> 𝜀∕2. By continuity, on (xn −δ,xn + δ), f(x) is either positive or negative in whole interval. But then,
|∫ xn−δxn+δf(x)dx| > 𝜀δ

∫ ℝ|f(x)|dx ≥∑ n=1∞∫ xn−δxn+δ|f(x)|dx ≥∑ n=1∞𝜀δ = ∞,

contradicts that f is integrable.

(a)
Prove that T∗T = I if T is an isometry.
(b)
Prove that if an isometry T is surjective then it is unitary and TT∗ = I.

(a)
Let T be an isometry on H. First, for f,g ∈H,
∥f −g∥2 = ∥f∥2 − 2 Re ⁡ ⟨f,g⟩ + ∥g∥2 = ∥Tf∥2 − 2 Re ⁡ ⟨Tf,Tg⟩ + ∥Tg∥2

Suppose that M is a σ-algebra in a set X and μ a (positive) measure on (X,M). For f ∈L1(μ), define a signed measure λ on (X,M) by λ(E) = ∫ Efdμ for E ∈M. Prove that
|λ|(E) = ∫ E|f|dμ.
Sol.
By Hahn decomposition and Jordan decomposition, λ = λ+ −λ− and |λ| = λ+ + λ− for two mutually singular positive measures λ±, where
λ+(E) = λ(E ∩X+),λ−(E) = −λ(E ∩X−),

where X± are disjoint subsets of X, such that λ(A) ≥ 0 (resp. ≤ 0) for any measurable A with A ⊂X+ (resp. ⊂X−). By the uniqueness of Hahn decomposition up to symmetric difference, f ≥ 0 on X+ a.e., and f < 0 on X− a.e.

Then we have

|λ|(E) = λ+(E) + λ−(E) = ∫ E∩X+fdμ −∫ E∩X−fdμ = ∫ E∩X+|f|dμ + ∫ E∩X−|f|dμ = ∫ E|f|dμ.

It can be shown without using Hahn decomposition. For a partition {Ej} of E,

∑ j |∫ Ejfdμ| ≤∑ j ∫ Ej|f|dμ = ∫ E|f|dμ

guarantees that |λ|(E) ≤∫ E|f|dμ. Conversely, E+ := {x ∈E : |f(x)|≥ 0} and E− := {x ∈E : |f(x)|< 0} form a measurable partition of E, with

|∫ E+fdμ| + |∫ E−fdμ| = ∫ E+|f|dμ + ∫ E−|f|dμ = ∫ E|f|dμ.

Hence the supremum of ∑ ⁡ j|λ(Ej)| over all measurable partitions of E is ∫ E|f|dμ.

Let F be an increasing function on [0, 1] with F(0) = 0 and F(1) = 1. Let μ be a Borel measure defined by μ((a,b)) = F(b−) −F(a+) and μ(0) = μ(1) = 0. Suppose that the function F satisfies a Lipschitz condition |F(x) −F(y)|≤A|x −y| for some A > 0. Prove that μ ≪m, where m is the Lebesgue measure on [0, 1].
Sol.
By Lipschitz condition, F is continuous, and hence μ((a,b)) = F(b) −F(a). Then immediately
μ (⊔ n=1∞(a n,bn)) = ∑ n=1∞μ((a n,bn)) ≤A∑ n=1∞(b n−an) = A⋅m (⊔ n=1∞(a n,bn)) .

That is, for open U ⊂ [0, 1], μ(U) ≤A ⋅m(U).

μ(E) ≤μ(V n) ≤A ⋅m(V n) < A ⋅m(E) + A∕n

for all n. Hence μ(E) ≤A ⋅m(E) = 0. Therefore μ ≪m is shown.

First, we will prove the statement m(O) = ∫ F−1(O)F′(x)dx

where the second equality is from absolute continuity. Therefore the statement in the hint is shown.

(F−1(G) ∩G′) ∖ (F−1(E) ∩P) = (F−1(G ∖E) ∩G′) ∪ (F−1(G) ∩ (G′∖P)) = (F−1(G ∖E) ∩ (P ∪ (G′∖P)) ∪ (F−1(G) ∩ (G′∖P)) = (F−1(G ∖E) ∩P) ∪ (F−1(G) ∩ (G′∖P)).

Since G ∖E has zero measure, there exists open On such that (G ∖E) ⊂On and m(On ∖ (G ∖E)) = m(On) ≤ 1∕n. Then

1 n ≥m(On) = ∫ F−1(On)F′(x)dx ≥∫ F−1(⋂iOi)F′(x)dx ≥∫ F−1(⋂iOi)∩P F′(x)dx

Sol.
Assume that f is neither injective nor constant. Then there is a complex number w such that f(z) = w has at least two solutions in D. As fn(z) −w converges to f(z) −w uniformly on K, by Hurwitz’s theorem, the number of zeros of f(z) −w is equal to the number of zeros of fn(z) −w for sufficiently large n.
